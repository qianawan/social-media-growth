# 自媒体起号 · 完整操盘手册

## 一、底层逻辑（先读这里）

内容运营的本质是**"关系感"的经营**。

> 你想吸引什么样的人，就用什么方式和他们相处。

三个核心认知：
1. **不完美人设 > 完美人设**：永远"正确"的账号无法和观众建立最深的连接。分享真实的不安、奇怪的念头、小情绪，才能让人觉得"这说的就是我"。
2. **涨粉选题 ≠ 变现选题**：涨粉靠共鸣+泛流量，变现靠明确的消费场景。两套内容要分开规划。
3. **吸引一部分人，而非讨好所有人**：所有人都想讨好 = 讨好0个人。

---

## 二、选赛道

### 原则
不追风口，做热爱的。长期能做下去的一定是你本身感兴趣、甚至有点"狂热"的方向。

### 三问自查法
- 平时你最愿意花时间研究什么？
- 别人最常问你什么？
- 你在哪个领域会忍不住疯狂输出？

### 不需要你是专家
只需要在某一个点上，**比一部分人懂得多一点点**就够了。

### 明确做号目的（必须先想清楚）
| 目的 | 策略重点 |
|------|----------|
| 变现 | 受众消费能力、接广告的内容结构 |
| 记录/社交 | 情绪共鸣、找同频的人 |
| 品牌/IP | 专业度建立、长期价值输出 |

---

## 三、人设定位 · 标签法

### 核心方法
一开始标签不用多，**2-3个就够**。

围绕这几个标签提问自己：
- 你是谁？（身份/职业）
- 什么外貌特征？（视觉记忆点）
- 擅长什么 / 有什么特殊经历？

### 示例
```
大学生 / 方圆脸 / 擅长自我暴露（讲自己的故事）
```

这三个标签可以发散出的选题：
- 大学生：升学考试、舍友关系、恋爱、情感困惑、个人成长、人生规划
- 方圆脸：变美、穿搭、护肤（变现方向）
- 自我暴露：情绪表达、真实故事（涨粉方向）

> 详细选题挖掘方法见 `references/topic-mining.md`

## 四、主页搭建

**主页优先级**：整体风格 + 简介 >> 头像 + 名字

### 简介公式
> 你是谁 + 你发什么 + 为什么值得关注

用户第一次刷到你的内容时，第一反应是："**这个账号能给我带来什么？**"——简介必须在3秒内回答这个问题。

### 风格一致性
- 封面色调、字体、构图风格保持统一
- 主页整体看起来像一个"系列"，而不是随机发布

## 五、内容策略

### 5.1 两类选题的分工

| 类型 | 作用 | 特征 | 举例 |
|------|------|------|------|
| **涨粉选题** | 扩大受众、建立连接 | 情绪共鸣、个人经历、观点输出、泛流量内容 | "我在XXX经历了什么"、情绪类故事 |
| **变现选题** | 承接品牌广告、产生消费需求 | 有明确"广告位"、天然对接消费品类 | 方圆脸变美、祛痘护肤、好物推荐 |

> 两类内容可以穿插发布，不要只发一种。

### 5.2 发布策略

**频率**：新账号前期持续发，哪怕没流量也别停。系统需要时间给你的账号打标签。但持续 ≠ 乱发，质量始终是第一位。

**时间**：根据目标用户活跃时间选择。一般规律：
- 学生党：午休（12-13点）、晚间（20-23点）
- 上班族：通勤时间、午休、睡前
- 周末和晚上整体流量更高

### 5.3 内容制作流程

1. **找对标账号** → 同赛道的头部账号 + 中腰部账号
2. **重点研究低粉爆款** → 低粉还能爆，说明内容本身够强，参考价值最高
3. **拆解爆款结构**：
   - 标题怎么写？用了什么钩子？
   - 封面怎么设计？
   - 开头前3秒/前两行是什么？
   - 评论区为什么有人互动？
4. **整理选题库** → 一次拆解至少产出一个月的选题

## 六、积累第一批粉丝

### 核心公式
**利他性 + 情绪共鸣** = 点赞、收藏、关注

用户关注一个账号，只有两个原因：
- "这个内容对我有用"（利他）
- "这说的不就是我吗"（共鸣）

### 钩子设计
在内容里留互动引导，让观众有话说：
- 提出一个问题让大家在评论区回答
- 留一个开放性的"等你来说"
- 制造争议点或选择题

> 互动越多 → 进入下一轮流量池 → 更多曝光

### 可持续增长的关键
单篇爆文不稳定，**持续出现爆文**才是真正做起来的标志。做到这一点需要：
- 持续输出（不断练习内容感觉）
- 找到自己的核心优势并放大
- 保持表达欲（不要为了做号而做号）

## 七、变现路径

### 接广告（国内主流变现方式）
优点：
- 直接从品牌方赚钱，不和粉丝产生利益纠缠
- 不用处理售后，省精力省人力

前提条件：内容里需要有清晰的"广告位"——即内容本身天然对接某类消费品。

> 更多变现路径详见 `references/monetization.md`

## 八、操盘清单（快速上手）

- [ ] 确定赛道（基于热爱，不追风口）
- [ ] 提炼2-3个标签，发散出30+选题
- [ ] 明确做号目的（变现 or 社交 or IP）
- [ ] 搭建主页（简介公式 + 风格统一）
- [ ] 找5-10个对标账号，重点分析低粉爆款
- [ ] 规划内容矩阵（涨粉选题 + 变现选题比例）
- [ ] 制定发布频率（新号建议每周3-5篇）
- [ ] 每篇内容设置互动钩子
- [ ] 每月复盘数据，迭代内容方向
