# 抖音 vs 小红书 推流机制（2026版）

## 一、核心底层逻辑

**抖音**：去中心化 + 赛马制。新视频发布后进入"冷启动池"，获得200-1000次曝光，若互动率达标则晋升至更高流量池，同类视频通过"赛马机制"竞争，优胜者获得更多推荐。粉丝量几乎不影响初始推流，完全看内容数据说话。

**小红书**：先审核收录 → 打标签匹配 → 阶梯式流量池 → CES评分定生死 → 搜推互通放大，形成"精准分拣+逐步扩面"的内容分发闭环。和抖音最大的区别是**搜索权重极高**——超过65%的流量来源于搜索，更像一个"内容搜索引擎"。

---

## 二、核心评分指标

**抖音**的权重排序（2026年更新后）：从24小时快速赛马转向7天长效流量，收藏率成为第一权重，健康分与AI标注成为硬门槛。具体指标：完播率 > 收藏率 > 评论 > 点赞。抖音官方内测数据显示，收藏率达到5%的内容，获得的流量扶持是收藏率1%内容的8倍。

**小红书** 用CES评分公式量化：CES = 点赞×1 + 收藏×1 + 评论×4 + 转发×4 + 关注×8。2026年更侧重互动质量：长评论、深度阅读、截屏等高质量互动权重更高，单纯点赞加分占比低。

---

## 三、最大机制变化（2026新规）

**抖音** 最核心的变化是从"24小时定生死"改成了7天考核：以往发布后24小时内就能获得大部分流量，现在平台改为一周内持续推流，意味着发布当天数据不佳并不代表作品质量差。实战案例印证了这一点——某知识类账号发布一条Python教程，首日播放不足1000，第4天播放量突然暴涨到12万，第7天累计超过100万播放。

**小红书** 的最大变化是算法从"关键词匹配"升级为"语义理解"：堆砌热门关键词的时代彻底过去了，只要内容表达自然、主题清晰，系统就能准确识别并推送给相关兴趣用户；靠关键词堆砌的"技术型"笔记在新算法下反而会被识别为低质内容。

另外小红书还新增了"搜推互通"：用户搜过某个关键词后，推荐页会优先推送相关内容，形成搜索和推荐流量的联动效应。

---

## 四、AI内容的处理方式

两个平台都在收紧，方向一致：

**抖音**：AI生成/辅助内容必须在标题、简介或评论区标注"AI生成"，未标注限流50%-80%，情节严重会下架或封禁带货权限。

**小红书**：不是AI工具不能用，而是必须在AI生成的基础上加入真实体验和独特视角，才能通过算法的真实度评估，"AI味浓"的笔记越来越难跑出来。

---

## 五、实操结论对比

| | 抖音 | 小红书 |
|---|---|---|
| 流量来源 | 推荐流为主 | 搜索流占65%+ |
| 最重要指标 | 收藏率 + 7天完播 | CES评分（评论/关注权重高） |
| 内容生命周期 | 7天长效赛马 | 可持续积累，长尾效应强 |
| 关键词策略 | 自然植入，不能硬堆 | 标题前18字含核心词，但禁止堆砌 |
| 新号冷启动 | 发布后30分钟内拉互动 | 垂直账号 + 养号5天再发内容 |
| 硬广 | 几乎被限流 | 同样限流，软广为主 |

---

两个平台趋势一致：**奖励真实、有价值、能留住人的内容，惩罚标题党、刷数据、AI水内容**。
