# 选题挖掘 · 深度指南

## 从标签发散选题

以"大学生 / 方圆脸 / 擅长自我暴露"为例：

### 标签1：大学生（身份标签）
**涨粉方向**：
- 升学/考研/保研经历
- 舍友关系（冲突、和解、奇葩故事）
- 校园恋爱与分手
- 情感困惑（暗恋、异地、出轨）
- 个人成长/迷茫期
- 人生规划（要不要考研、要不要出国）
- 实习踩坑记录

**变现方向**：
- 学习工具好物推荐
- 宿舍好物
- 平价穿搭
- 考试备考资料

### 标签2：方圆脸（外貌标签）
**涨粉方向**：
- 方圆脸显瘦/显尖的技巧（引发共鸣）

**变现方向**：
- 方圆脸适合的发型/妆容/穿搭 → 接美妆、护肤、服装广告
- 祛痘/改善肤质记录 → 接护肤品广告

### 标签3：自我暴露（内容风格标签）
**涨粉方向**（几乎全是涨粉选题）：
- 我曾经最低谷的一段时间
- 我做过最后悔的一件事
- 我谈过最不成熟的一段感情
- 我曾经最讨厌自己的什么
- 我和XXX的关系终于结束了

## 低粉爆款分析模版

拆解一篇低粉爆款时，记录以下维度：

```
账号粉丝数：____
内容类型：图文 / 视频
点赞数：____ 收藏数：____ 评论数：____
互动率（点赞/曝光）：约____

【标题分析】
使用了什么钩子？（数字/问句/情绪词/稀缺感）

【封面分析】
色调：____
有没有文字？文字说了什么？

【开头（前3秒/前两行）】
是什么让人看下去了？

【内容结构】
起：____
承：____
转：____
合：____

【互动钩子】
结尾是怎么引导评论的？

【为什么能爆】（你的判断）
____
```

## 选题库管理建议

- 用备忘录/Notion建立选题库
- 分两个列表：涨粉选题 / 变现选题
- 每次刷到好内容，立刻记下来
- 每周固定做一次选题会（给自己30分钟）
- 目标：永远保持库里有15+备用选题
